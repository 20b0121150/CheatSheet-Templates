This Linux Cheatsheet For You!
  1. To go up one level of the directory tree. (Change into the parent directory.) 
    cd ..
  2. Go to the $HOME directory 
    cd
  3. Change to the /etc directory 
    cd /etc
  4. Display Linux system information 
    uname -a
  5. Display kernel release information 
    uname -r
  6. Show how long the system has been running + load 
    uptime
  7. Show system host name 
    hostname
  8. Display all local IP addresses of the host. 
    hostname -I
  9. Show system reboot history 
    last reboot
  10. Show the current date and time 
    date
  11. Show this month's calendar 
    cal
  12. Who you are logged in as 
    whoami
  13. Display messages in kernel ring buffer 
    dmesg
  14. Display CPU information 
    cat /proc/cpuinfo
  15. Display memory information 
    cat /proc/meminfo
  16. Display free and used memory ( -h for human readable, -m for MB, -g for GB.) 
    free -h
  17. List all files in a long listing (detailed) format 
    ls -al
  18. Create a directory 
    mkdir directory
  19. Display the present working directory 
    pwd
  20. Remove (delete) file 
    rm file
  21. Remove the directory and its contents recursively 
    rm -r directory
  22. Force removal of file without prompting for confirmation 
    rm -f file
  23. Forcefully remove directory recursively 
    rm -rf directory
  24. Copy file1 to file2 
    cp file1 file2
  25. Copy source_directory recursively to destination. If destination exists, copy source_directory into destination, otherwise create destination with the contents of source_directory. 
    cp -r source_directory destination
  26. Rename or move file1 to file2. If file2 is an existing directory, move file1 into directory file2 
    mv file1 file2
  27. Create symbolic link to linkname 
    ln -s /path/to/file linkname
  28. Create an empty file or update the access and modification times of file. 
    touch file
  29. View the contents of file 
    cat file
  30. Browse through a text file 
    less file
  31. Display the first 10 lines of file 
    head file
  32. Display the last 10 lines of file 
    tail file
  33. Display the last 10 lines of file and "follow" the file as it grows. 
    tail -f file
  34. Display PCI devices 
    lspci -tv
  35. Display USB devices 
    lsusb -tv
  36. Display DMI/SMBIOS (hardware info) from the BIOS 
    dmidecode
  37. Show info about disk sda 
    hdparm -i /dev/sda
  38. Perform a read speed test on disk sda 
    hdparm -tT /dev/sda
  39. Test for unreadable blocks on disk sda 
    badblocks -s /dev/sda
  40. Display and manage the top processes 
    top
  41. Interactive process viewer (top alternative) 
    htop
  42. Display processor related statistics 
    mpstat 1
  43. Display virtual memory statistics 
    vmstat 1
  44. Display I/O statistics 
    iostat 1
  45. Display the last 100 syslog messages (Use /var/log/syslog for Debian based systems.) 
    tail -100 /var/log/messages
  46. Capture and display all packets on interface eth0 
    tcpdump -i eth0
  47. Monitor all traffic on port 80 ( HTTP ) 
    tcpdump -i eth0 'port 80'
  48. List all open files on the system 
    lsof
  49. List files opened by user 
    lsof -u user
  50. Display free and used memory ( -h for human readable, -m for MB, -g for GB.) 
    free -h
  51. Execute "df -h", showing periodic updates 
    watch df -h
  52. Display the user and group ids of your current user. 
    Display the user and group ids of your current user.
  53. Display the last users who have logged onto the system. 
    last
  54. Show who is logged into the system. 
    who
  55. Create a group named "test". 
    groupadd test
  56. Create an account named john, with a comment of "John Smith" and create the user's home directory. 
    useradd -c "John Smith" -m john
  57. Delete the john account. 
    userdel john
  58. Add the john account to the sales group 
    usermod -aG sales john
  59. Display your currently running processes 
    ps
  60. Display all the currently running processes on the system. 
    ps -ef
  61. Kill process with process ID of pid 
    kill pid
  62. Kill all processes named processname 
    killall processname
  63. Start program in the background 
    program &
  64. Display stopped or background jobs 
    bg
  65. Brings the most recent background job to foreground 
    fg
  66. Brings job n to the foreground 
    fg n
  67. File Permissions 
    
            PERMISSION      EXAMPLE

            U   G   W
            rwx rwx rwx     chmod 777 filename
            rwx rwx r-x     chmod 775 filename
            rwx r-x r-x     chmod 755 filename
            rw- rw- r--     chmod 664 filename
            rw- r-- r--     chmod 644 filename

            # LEGEND
              U = User
              G = Group
              W = World

              r = Read
              w = write
              x = execute
              - = no access
  68. Search for a package by keyword. 
    yum search keyword
  69. Install package. 
    yum install package
  70. Display description and summary information about package. 
    yum info package
  71. Install package from local file named package.rpm 
    rpm -i package.rpm
  72. Remove/uninstall package.
  73. yum remove package
  74. Install software from source code. 
    
            tar zxvf sourcecode.tar.gz
            cd sourcecode
            ./configure
            make
            make install
  75. Search for pattern in file 
    
            grep pattern file
  76. Search recursively for pattern in directory 
    grep -r pattern directory
  77. Find files and directories by name 
    locate name
  78. Find files in /home/john that start with "prefix". 
    find /home/john -name 'prefix*'
  79. Find files larger than 100MB in /home 
    find /home -size +100M
  80. Show free and used space on mounted filesystems 
    df -h
  81. Show free and used inodes on mounted filesystems 
    df -i
  82. Display disks partitions sizes and types 
    fdisk -l
  83. Display disk usage for all files and directories in human readable format 
    du -ah
  84. Display total disk usage off the current directory 
    du -sh
