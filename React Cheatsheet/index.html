Here is the React Cheatsheet For You !

  1. Importing React 

    import React from 'react'
import ReactDOM from 'react-dom'

  2. Components types


    • Class componets have state. 
      class SimpleComponent extends React.Component {
    render () {
        return <div className='myClassName'>
        This is simple component
        </div>
    }
}

    • Pure components are performance-optimized and doesn’t rerender if props/state hasn’t changed. 
      import React, {PureComponent} from 'react'

class PureComponentExample extends PureComponent {
}

    • Functional components have no state. Also, their props are passed as the first parameter to a function. 
      function FunctionalExample ({ yourProps }) {
    return <div className='yourClassName'>
      Hello {yourProps}</div>
  }

  3. States

    constructor(props) {
    super(props)
    this.state = { someVariable: undefined }
}
    • Updating states in class component.This is recommended way, do no mutate state directly. 
      this.setState({ someVariable: 'mutable' })
    • Functional components have no state, but we can achieve a similar effect with useState hook. 
      const [someVariable,setSomeVariable] = React.useState(undefined)

setSomeVariable('mutable')

  6. DOM Events

    class MyComponent extends Component {
    constructor(props){
        super(props)
        this.state={value:null}
    }
    render () {
        <input type="text"
        value={this.state.value}
        onChange={event => this.onChange(event)} />
            }
                  
    onChange (event) {
        this.setState({ value: event.target.value })
    }
}

  7. References

    class MyComponent extends Component {
    render () {
        return ( <div>
            <input ref={el => this.input = el} />
        </div>)
        }
                  
    componentDidMount () {
        this.input.focus()
    }
}
